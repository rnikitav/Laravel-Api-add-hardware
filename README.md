## Настройка проекта

- `composer install`
- npm install
- настроить `.env` файл
- `php artisan key:generate`
_____________
###Настройка БД
1) создать БД локально или на сервере и указать ее в .env файле 
2) php artisan migrate

_____________

## Запуск приложения

* php artisan serve
______
### При необходимости 
*  npm run watch
